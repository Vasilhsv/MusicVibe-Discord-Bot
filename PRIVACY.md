# Privacy Policy for MusicVibe

### 1. Data Collection & Logging
MusicVibe collects limited technical logs to ensure service stability and security.
* **What we log:** Usernames, command names (e.g., `/play`, `/join`), and Server IDs.
* **Purpose:** Debugging, preventing abuse, and improving bot performance.
* **Retention:** Logs are stored temporarily on a private server and are automatically cleared periodically (every 7-30 days).
* **No Personal Data:** We do **not** log chat messages (outside of direct commands), voice recordings, or IP addresses.

### 2. Real-time Processing
We only process commands in real-time to provide the requested music services. Outside of the technical logs mentioned above, no user data is cached or saved after a command is executed.

### 3. Third-Party Services
Our bot interacts with **YouTube** and **SoundCloud** APIs to stream audio. No user-identifiable data is shared with these third parties during the streaming process.

### 4. Infrastructure & Security
The bot is hosted on a **private, dedicated server**. 
* No data is shared with third-party cloud hosting providers.
* All session memory is cleared automatically upon bot restart. 
* We prioritize the security and privacy of the servers where MusicVibe is present.
